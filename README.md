# 🎨 ARTORIUM

> Plateforme de découverte artistique propulsée par l'API du **Art Institute of Chicago**.


🔗 **[Voir le site en ligne](https://moad-adriouche.github.io/artorium.github.io)**

---

## Description

ARTORIUM permet d'explorer plus de **100 000 œuvres d'art** issues de la collection ouverte du Art Institute of Chicago. Interface moderne, responsive et disponible en **Français, Anglais et Arabe**.

## Technologies

- HTML5 / CSS3 / JavaScript Vanilla
- [Art Institute of Chicago API](https://api.artic.edu/docs/) — publique et gratuite
- `localStorage` pour la persistance de la langue

## Fonctionnalités

- 🖼️ Galerie dynamique chargée via API
- 🔍 Recherche par mot-clé et filtres (Peintures, Sculptures, Photographies)
- 📄 Modal de détail pour chaque œuvre
- 🌐 Interface multilingue — FR / EN / AR
- 📬 Formulaire de contact avec validation
- 📱 Design responsive et pagination

## Lancer le projet

```bash
git clone https://github.com/moad-adriouche/artorium.github.io.git
cd artorium.github.io
# Ouvrir index.html dans le navigateur
```

> ⚠️ Une connexion Internet est requise pour charger les œuvres via l'API.
